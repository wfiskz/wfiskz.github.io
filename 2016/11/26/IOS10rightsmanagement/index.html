#iOS 10的权限管理

##1.不添加权限app会闪退
我们开发应用时，经常会用到访问相片，通讯录等，在iOS9及以前苹果对这个权限管理并不是特别严格，但在iOS10你必须添加对应的权限，否则就会闪退。
错误如下：
This app has crashed because it attempted to access privacy-sensitive data without a usage description.
The app’s Info.plist must contain an NSCameraUsageDescriptionkey with a string value explaining to the user how the app uses this data.
